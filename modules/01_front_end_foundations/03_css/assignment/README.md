# Assignment: The Skin

Now we make it look good.

## Setup

1.  Continue in your `profile-card-project` folder.
2.  Create a file named `style.css`.
3.  Link it in your HTML `<head>`:
    ```html
    <link rel="stylesheet" href="style.css" />
    ```

## Instructions

Use CSS to style your card. Try to match this design:

1.  **Card**:

    - Add a border and some padding.
    - Add a `border-radius` to round the corners.
    - Center the text (`text-align: center`).
    - Limit the width (e.g., `width: 300px`) and center the card on the page (hint: use Flexbox on the `body`).

2.  **Image**:

    - Make it a circle (`border-radius: 50%`).
    - Set a fixed width/height (e.g., `100px`).

3.  **Button**:
    - Change the background color to blue and text color to white.
    - Remove the default border.
    - Add some padding.

## Deliverable

Your `index.html` should now look like a proper component, not a Word document.
